🚀 Travel Booking System (WanderWise)

A full-stack Travel Booking System built using React, Node.js, Express, and MySQL.
This project allows users to explore destinations, book trips, and manage their bookings with authentication and user-specific data handling.


✨ Features
🔐 User Registration & Login
🔑 Forgot Password & Reset Password
🌍 Browse travel destinations
🔎 Search & Pagination for destinations
📦 Book travel destinations
👤 User-specific booking history
🗑️ Delete bookings feature
⚡ REST API integration (Frontend + Backend)
📱 Responsive UI with modern design

🛠️ Tech Stack
Frontend: React.js, Axios, Tailwind CSS
Backend: Node.js, Express.js
Database: MySQL
API: RESTful APIs

frontend/   → React UI
backend/    → Node.js API
database/   → MySQL schema


🚀 How It Works
User registers or logs in
Browses destinations
Selects a destination and books it
Booking is stored in MySQL with user ID
User can view or delete their bookings


🎯 Future Improvements
JWT Authentication (secure login system)
Payment gateway integration
Admin dashboard (manage destinations & bookings)
Email notifications for bookings
Cloud deployment (Vercel + Render)



📌 Purpose

This project is built for learning full-stack development, API integration, authentication flow, and real-world booking system architecture.
